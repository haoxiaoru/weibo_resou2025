# 2025年微博热搜AI趋势分析 - 交互式报告

> 基于364天微博热搜数据的AI趋势深度分析

## 📊 报告概览

- **1,423次** 泛AI热搜提及
- **21个** AI产品深度分析  
- **DeepSeek** 全年霸榜410次
- **87.4%** 年度覆盖率

## 🚀 如何查看报告

## 💡 交互功能

1. **点击产品查看详情** - 左侧点击任意AI产品，右侧显示所有热搜内容
2. **搜索功能** - 右上角搜索框快速定位产品
3. **排序功能** - 可按热度或字母顺序排序
4. **时间排序** - 热搜内容按时间倒序排列（最新在上）
